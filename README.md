# Описание работы

В данном задании мной были повторены основы работы с базами данных, используя язык `PostgeSQL`. Решенные задания
охватывают создание, наполнение и манипуляцию данными в различных базах данных

Была проведена работа с **четырьмя** различными базами данных

Каждая из баз данных включает в себя этапы создания таблиц, наполнения их данными, а также решение задач разной
сложности, что позволило на практике применить полученные из модулей знания

Каждая из представленных баз данных сопровождается скриптом, который создает таблицы базы данных и заполняет их
тестовыми данными
